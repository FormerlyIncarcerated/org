## 🚀 **Deployment & Setup**

### **Quick Start**
```bash
# Clone and install
git clone https://github.com/FormerlyIncarcerated/org.git
cd org
bun install

# Environment setup
cp .env.example .env.local
# Configure environment variables

# Database setup
bun db:setup

# Start development
bun dev
```

### **Production Deployment**
```bash
# Build for production
bun build

# Deploy to Vercel
vercel --prod

# Set up community infrastructure
bun community:setup
```
